### Descripción general del programa:
Este programa ofrece un menú principal con tres opciones: detección de colores, lectura de sensores y salir del programa. El usuario puede seleccionar la opción deseada para realizar tareas específicas relacionadas con la detección de colores y la lectura de sensores.

### Funciones del programa:

1. **`main_menu()`**:
   - **Descripción**: Muestra el menú principal del programa.
   - **Salidas**:
     - Imprime las opciones disponibles para el usuario, incluyendo detección de colores, lectura de sensores y salir del programa.

2. **`run_color_detection()`**:
   - **Descripción**: Maneja la interfaz de usuario para la detección de colores.
   - **Dependencias**: Importa y utiliza el módulo `colores` (que debe estar definido previamente en otro archivo).
   - **Funcionamiento**:
     - Presenta un submenú específico para la detección de colores.
     - Permite al usuario iniciar la detección de colores o volver al menú principal.
     - Utiliza el método `capture_objects()` del módulo `colores` para ejecutar la detección.

3. **`run_sensor_readings()`**:
   - **Descripción**: Maneja la interfaz de usuario para la lectura de sensores.
   - **Dependencias**: Importa y utiliza el módulo `dht` (que debe estar definido previamente en otro archivo).
   - **Funcionamiento**:
     - Presenta un submenú específico para la lectura de sensores.
     - Permite al usuario iniciar la lectura de los sensores o volver al menú principal.
     - Utiliza la función `main()` del módulo `dht` para realizar la lectura de los sensores.

4. **Bloque principal `if __name__ == "__main__"`**:
   - **Descripción**: Controla el flujo principal del programa.
   - **Funcionamiento**:
     - Ejecuta un bucle que mantiene el programa en ejecución hasta que el usuario decida salir.
     - Permite la navegación entre las diferentes funciones basadas en la entrada del usuario.

### Consideraciones adicionales:
- Asegúrate de que los módulos `colores` y `dht` estén correctamente definidos y disponibles en el entorno de ejecución, ya que el programa depende de estos para las funcionalidades de detección de colores y lectura de sensores.
- El programa utiliza la entrada de usuario para navegar entre las opciones, lo que requiere que el usuario introduzca valores válidos según las instrucciones dadas.
